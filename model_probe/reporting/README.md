# 交互式分析报告系统设计文档

## 🎯 目标

让用户能够**直观、交互式地理解**模型内部工作机制。

## 设计原则

### 1. 分层展示 (Progressive Disclosure)
```
概览 (1页) → 关键发现 → 详细分析 → 原始数据
  ↓           ↓           ↓          ↓
 30秒       2分钟       10分钟     深入研究
```

### 2. 问题驱动 (Question-Driven)
不是"我们有什么数据"，而是"回答什么问题"：
- 这个模型在做什么？
- 知识存储在哪里？
- 不同层关注什么？
- 如果我修改会怎样？

### 3. 可视化优先 (Visualization First)
- 用颜色编码突出重要信息
- 交互式图表，可以hover看详情
- 动画展示变化过程

### 4. 故事化叙述 (Storytelling)
用"故事线"引导用户：
```
第一章：模型概览 - "这是什么？"
第二章：层的行为 - "每层在做什么？"
第三章：注意力机制 - "它在关注什么？"
第四章：知识定位 - "知识存在哪？"
第五章：交互探索 - "如果...会怎样？"
```

## 核心功能

### 1. 智能概览页面
```html
┌─────────────────────────────────────────┐
│  📊 GPT-2 模型分析报告                   │
├─────────────────────────────────────────┤
│                                         │
│  ⭐ 关键发现（3个最重要的发现）          │
│  • 第6-8层编码语法知识                  │
│  • 第9-11层编码语义知识                 │
│  • 注意力头5专门处理指代关系             │
│                                         │
│  📈 快速统计                             │
│  ┌─────┬─────┬─────┬─────┐             │
│  │层数 │参数 │准确率│... │             │
│  └─────┴─────┴─────┴─────┘             │
│                                         │
│  🎯 点击下方卡片深入了解：               │
│  ┌──────┐ ┌──────┐ ┌──────┐           │
│  │层分析│ │注意力│ │知识  │           │
│  └──────┘ └──────┘ └──────┘           │
└─────────────────────────────────────────┘
```

### 2. 交互式层分析
- **热力图**：每层每个探针的准确率
- **点击展开**：点击某层查看详细信息
- **对比模式**：选择2-3层进行对比
- **时间线**：展示信息如何逐层累积

### 3. 注意力可视化
- **交互式注意力图**：hover查看权重
- **头聚类**：相似的头自动分组
- **模式识别**：自动标注常见的注意力模式
- **动画**：展示注意力如何随token变化

### 4. 知识定位
- **神经元重要性排序**：哪些神经元最重要
- **层级知识图谱**：知识在不同层的分布
- **干预模拟**："如果禁用这些神经元会怎样？"

### 5. 交互探索
- **参数调节**：实时调整输入看结果
- **What-if分析**：修改模型权重观察影响
- **对比工具**：对比两个模型的差异

## 技术实现

### 前端技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **原生JavaScript** - 交互逻辑
- **Plotly.js** - 交互式图表
- **D3.js** - 复杂可视化

### 后端技术栈
- **Jinja2** - HTML模板引擎
- **Plotly** - Python图表生成
- **Markdown** - 报告内容生成

## 数据流程

```
分析脚本 → JSON数据 → 报告生成器 → HTML报告 → 浏览器展示
              ↓                            ↓
         结构化数据                    交互式界面
```

## 使用示例

```python
from model_probe.reporting import ReportBuilder

# 1. 运行分析
wrapper = ModelWrapper(ModelConfig("gpt2"))
wrapper.load_model()

analyzer = ModelAnalyzer(wrapper)
results = analyzer.analyze_all()

# 2. 生成报告
builder = ReportBuilder(results)
report = builder.build(
    title="GPT-2 模型分析报告",
    output_dir="outputs/interactive_report",
    interactive=True
)

# 3. 在浏览器中打开
# report.open_in_browser()
```

## 输出格式

生成一个自包含的HTML文件，可以直接在浏览器中打开：
- 单文件，无需服务器
- 包含所有CSS和JS
- 数据嵌入在HTML中
- 支持离线查看
