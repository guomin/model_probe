# Model Probe 愿景与目标

## 核心使命

**揭开模型黑盒，让 AI 白化与透明化**

从"能用就行"到"真正掌握"，构建可解释、可控制、可改造的神经网络理解框架。

## 目标层次

```
理解 → 掌握 → 利用 → 改造 → 更深理解
  ↑                          │
  └──────────────────────────┘
           持续迭代
```

### 1. 理解
- 知识存储在哪些层/神经元
- 模型如何处理特定信息
- 注意力在关注什么

### 2. 掌握
- 预测模型行为
- 预判失败场景
- 控制信息流动

### 3. 利用
- 优化模型性能
- 指导模型设计
- 辅助特征工程

### 4. 改造
- 精准编辑知识
- 修复错误认知
- 注入新能力

## 理解的目的

| 目的 | 说明 |
|------|------|
| 可控 | 能预测行为，不出意外 |
| 可靠 | 知道何时失败，可靠部署 |
| 可信 | 透明才有信任 |
| 可改 | 出了问题能定位修复 |
| 合规 | 满足法律要求（GDPR等） |

## 技术路线

### 探针体系

**按训练分类：**
- 训练探针：线性探针、MLP探针
- 非训练探针：直接分析激活统计

**按方式分类：**
- 静态分析：直接分析参数/权重
  - 权重统计分析（均值、标准差、稀疏度）
  - **矩阵分析**（秩、条件数、奇异值分解、特征谱）
  - 层相似度分析（余弦相似度、CKA）
- 动态分析：Hook捕获推理过程的中间激活
  - 激活值统计分析
  - **激活矩阵分析**（表示相似度、信息流）

**按粒度分类：**
- Token级：单个token表示
- Layer级：整层输出
- Head级：Attention head分析

### 核心模块

```
ModelProbe/
├── 探测 (Probe)       # 找出知识在哪
├── 分析 (Analysis)    # 解释知识是什么
├── 可视化 (Visualize) # 让人类能看懂
├── 编辑 (Editor)      # 可干预、可控修改
└── 验证 (Verify)      # 确认改动效果
```

## 设计原则

1. **模块化** - 各模块可独立使用
2. **通用性** - 支持 Transformer/CNN/RNN 等多种架构
3. **可扩展** - 易于添加新的探针类型和分析方法
4. **实用性** - 能在有限资源下运行（如 16G 显存）

## 多角度分析体系

### 分析维度

```
┌─────────────────────────────────────────────────────────┐
│                   模型理解的多个角度                    │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  1. 权重统计分析                                         │
│     • 分布特性（均值、方差、稀疏度）                      │
│     • 参数量分布                                         │
│     • 应用：剪枝、量化                                    │
│                                                          │
│  2. 矩阵分析 ⭐ NEW                                     │
│     • 秩（信息容量）                                      │
│     • 条件数（数值稳定性）                                │
│     • 奇异值分解（SVD）                                   │
│     • 特征谱分析                                         │
│     • 应用：压缩、训练监控、架构设计                        │
│     📖 详见：[矩阵分析方法](./02_矩阵分析方法.md)        │
│                                                          │
│  3. 层相似度分析                                         │
│     • 静态：权重相似度                                    │
│     • 动态：激活相似度                                    │
│     • 应用：架构搜索、知识蒸馏                             │
│                                                          │
│  4. 激活分析                                             │
│     • Hook捕获中间层                                     │
│     • 统计特性                                           │
│     • 应用：理解功能、定位知识                             │
│                                                          │
│  5. 注意力分析                                           │
│     • 注意力模式                                         │
│     • 头聚类                                             │
│     • 应用：可解释性                                     │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 综合应用

```
完整分析流程：

1️⃣  静态分析（无需推理）
    ├─ 权重统计：快速了解参数分布
    ├─ 矩阵分析：理解数学结构
    └─ 层相似度：发现层关系

2️⃣  动态分析（需要推理）
    ├─ 激活分析：理解实际行为
    ├─ 注意力分析：看关注什么
    └─ 探针分析：定位知识

3️⃣  综合理解
    ├─ 对比静态vs动态
    ├─ 发现模式
    └─ 形成假设

4️⃣  验证假设
    ├─ 编辑模型
    ├─ 重新分析
    └─ 确认效果
```
